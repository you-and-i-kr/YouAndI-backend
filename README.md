* 23.10 ~ 23.11 (1개월) 


### ⚙️ 개발 환경
- **JAVA 11**
- **IDE** : IntelliJ
- **Framework** : Springboot 2.7.14
- **Database** : MySQL
- **ORM** : JPA
- **DOCS** : Swagger
- **LOGGING** : Slf4J

### ⚒️ 사용 기술 
- Spring Web MVC
- Spring Security + JWT
- Spring JPA
- Spring Logging
- PostMan(Test)
- API Swagger

<br>

## 📌 주요 기능
### 홈(HOME)
-새로운 활동에 대한 알림을 나타냄
-프로필 사진과 만난 날짜를 확인

### 채팅(chatting)
- 실시간으로 사진과 메세지를 주고 받음
  
### 앨범(Album)
- 사진 영상 업로드
- 각 게시물에 댓글 달기
- 댓글 삭제
  
### 캘린더(Calendar)
- 기념일,일정 추가 관리
- 수정 버튼
- 삭제 버튼 

### 환경설정(Setting)
- 기록장 이름 표시
- 기록장 이름 수정
- 회원탈퇴 기능
